Main Web Projects
CollectIT

CollectIT is my latest full stack IT Asset Management application (MERN stack ) and currently in it's final stages of development. This application will be open sourced once complete. More details about the apps features and technologies can be found on the github link's README.

campTrack

The idea behind campTrack is to allow travel enthusiats and adventurers to plot camping/trekking spots/trails on the map that other users can refer to while planning a trek or camp outing. Users can view Pin/Plot information about the camp site along with a description and picture uploaded by the author. Comments can also be added to pins. It uses React with Hooks on the frontend and GRAPHQL (APOLLO SERVER) and NODE along with MONGODB in the backend. It uses a react library for google OAuth sign in and multiple small libraries that helped me put the app together. The front-end application is deployed to NOW here and the GRAPHQL backend to HEROKU and is currently live.

Swapz

The idea behind this project was to create a platform where users can exchange or sell items ranging from simple goods to cars etc. This is a full stack application that uses JQUERY in the frontend and nodejs along with a SQL ORM in the backend. It uses JWT web tokens for authencation and bcrypt for hashing passwords. The application is deployed to Heroku here

GoalKeeper

Goalkeeper is my latest full stack application built with PHP and Laravel along with React in the frontend that allows users to set their goals for the day and keep a track of them. Users can register, login, create, update and delete goals. The main goal behind the application is to demonstrate working knowledge of PHP with the Laravel framework with CRUD operations and React in the frontend. The application is deployed to Heroku here

Flight11

This project is an online travel agency that allows users to get flight and ticketing information to book flight tickets. It uses JQuery in the front-end and consumes 3 APIs . A demo of the application can be seen in the README of the applications github page Here.
